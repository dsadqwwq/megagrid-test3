Testnet Demo

Paint tiles in real time.

Connect your wallet to the MegaETH testnet and color tiles. Changes are batched and broadcast as on-chain events.

How it works
1

Connect wallet

MegaETH testnet recommended. Make sure you have test funds.

2

Click to paint

Clicks are buffered for one second. Then we send one on-chain transaction with all changes.

3

Watch it update live

The app listens to events and updates everyone’s grid in real time.
